Het Grote Plaatje

Deze game is een 2D platformer gebouwd in Pygame. Het unieke aspect is de "kameleon" mechaniek (klimmen op muren/plafonds) en de aangepaste tools die we hebben gebouwd om development te versnellen.

Core Loop (`initial.py`)

Beheert de game loop, Death Counter, Pauze Menu, en render logic.

Fysica (`player.py`)

Beweging, zwaartekracht, wall-jumps, en de nieuwe Instant Reset functionaliteit.

Tools (`LevelBuilder/`)

Web-based tool om levels te tekenen en exporteren naar Python code. Inclusief Flood Fill en Import.

UI (`menus.py` & `standard_use.py`)

Pauze menu's, HUD (Death Counter, Health), en laadschermen.
