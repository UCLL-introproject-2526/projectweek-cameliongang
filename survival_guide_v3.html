Het Grote Plaatje

Deze game is een 2D platformer gebouwd in Pygame. Het unieke aspect is de "kameleon" mechaniek (klimmen op muren/plafonds) en de aangepaste tools die we hebben gebouwd om development te versnellen.

Core Loop (`initial.py`)

Nu met Asyncio ondersteuning voor web-browsers. Beheert states en de 60 FPS loop.

Fysica (`player.py`)

Verbeterde wall-climb detectie en Anti-Clip logica om bugs in hoeken te voorkomen.

Web Build (`game_itch/`)

Geoptimaliseerd voor itch.io via Pygbag. Draait op WebAssembly.

Optimalisatie (`level.py`)

Image Caching systeem vermindert laadtijden drastisch voor grote levels.
